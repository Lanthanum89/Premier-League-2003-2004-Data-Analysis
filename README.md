# Premier League 2003/2004 Data Analysis

This Jupyter Notebook provides an in-depth analysis of the English Premier League 2003/2004 season using Python, pandas, and matplotlib. The project explores team and match statistics, visualizes trends, and highlights key performances throughout the season.

## Contents

- **Data Loading & Preparation:**  
  Loads match data from `Prem0304.csv` and prepares it for analysis.

- **Team Performance Analysis:**  
  Calculates league tables, points, wins, draws, losses, and compares home vs. away performance for each team.

- **Points Progression:**  
  Visualizes how the top teams accumulated points over the season.

- **Streaks:**  
  Identifies the longest winning, losing, and unbeaten runs for each team.

- **Home vs. Away Comparison:**  
  Compares points earned at home and away for all teams.

- **Match Analysis:**  
  Highlights the biggest wins, highest scoring matches, and surprising upsets.

- **Heatmaps:**  
  Visualizes home wins, away wins, and draws between all pairs of teams.

- **Correlation Analysis:**  
  Examines the relationship between goals scored/conceded and final league position.

- **Clean Sheets:**  
  Shows which teams kept the most clean sheets.

- **Top Performers:**  
  Bar charts for top teams by goals scored, fewest goals conceded, and most points.

- **Team-Specific Visualizations:**  
  Detailed season breakdowns for Arsenal, Wolves, and Leeds, including goals, points progression, and result breakdowns.

- **Pie Charts:**  
  Win/Draw/Loss percentage pie charts for Arsenal, Wolves, and Leeds.

## How to Use

1. **Open the Notebook:**  
   Launch `Premier-League-2003-2004.ipynb` in Jupyter or VS Code.

2. **Run All Cells:**  
   Execute the notebook cells in order to load the data and generate all analyses and visualizations.

3. **Explore:**  
   Review the outputs, charts, and tables for insights into the 2003/2004 Premier League season.

4. **Modify:**  
   You can adapt the code to analyze other teams or add new visualizations as needed.
